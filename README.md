# socket.io-chat-example

## INSTALATION

* Run `npm install` in backend directory.
* Run `npm install` in frontend directory.
* Install **MySQL 2** in your PC, and import the database running the **data_base.sql** file.

## CONFIGURATION

* Edit the **.env** file, in the **backend** directory with your database information.

## INITIALIZATION

* Run `npm start` in the **backend** directory.
* Run `npm run serve` in the **frontend** directory.
